Thanks for adding Simple Excel Viewer 👋

You can now preview spreadsheets directly inside Trello cards.

  1. Attach an Excel or CSV file to any card (XLSX, XLS, CSV, XLSM, XLSB, ODS).
  2. Open the card — an Excel Files section appears with a Preview button.
  3. Click Preview to open the file in a read-only, in-card viewer.

Rename and Delete actions require a one-time authorization the first time you use them. You can disconnect any time from the Power-Up Settings.
