# CHANGELOG

## Версія: hardening for scale (гілка feature/hardening-for-scale)

Дата: 2026-05-12

## Що змінилось

### 1. Обмеження розміру файлу в /api/proxy (25 MB)

Сервер тепер відмовляє у проксі-завантаженні файлів більших за 25 MB і повертає
HTTP 413 з повідомленням замість того щоб витрачати пам'ять/трафік на надмірні
файли.

- Якщо upstream віддав заголовок Content-Length > 25 MB — відмова до завантаження.
- Якщо Content-Length відсутній (chunked) і фактичний розмір буфера перевищив 25 MB
  після завантаження — теж відмова.

Це захищає:
- Vercel-функцію від OOM (memory) при дуже великих файлах
- Hobby-план Vercel від швидкого вичерпання 100 GB/місяць трафіку
- Користувача від нескінченного очікування на великий файл

### 2. Тайм-аути на стороні клієнта

- **Preview**: 12 секунд (трохи більше за 10-секундний таймаут Vercel функції).
  Якщо за цей час файл не завантажився — побачите "Request timed out. The file
  may be too large or the server is slow." замість безкінечного спінера.
- **Download**: 30 секунд (для великих файлів — дозволяє довший час).

### 3. Детальні повідомлення про помилки

Коли проксі повертає помилку (401, 413, 500 тощо), інтерфейс тепер показує
причину з тіла відповіді сервера, а не просто HTTP-код.

Приклади того, що ви побачите:
- "413: File is 30.2 MB; preview limit is 25 MB."
- "401: invalid token" (теоретично — auth tokens мали б працювати)
- "Network error: Failed to fetch"
- "Request timed out..."

Раніше було просто "HTTP 413: Payload Too Large".

## Що тестувати на preview-URL

Стандартний потік:
[ ] Звичайний xlsx-файл відкривається в Preview як раніше — без регресій.
[ ] CSV-файл відкривається — без регресій.
[ ] Файл сф-1530_2 Олбрізсервіс 28.04.2026 (3).xlsx відкривається швидко
    (фікс з main, він не повинен зламатись).
[ ] Багатолистовий xlsx — перемикання вкладок аркушів працює.
[ ] Кнопка Download для звичайного файлу — файл скачується на диск.
[ ] Кнопка Rename — діалог prompt, перейменування зберігається.
[ ] Кнопка Delete — confirm діалог, видалення працює.
[ ] Меню "⋯" відкриває popup з 3-ма пунктами: Download / Rename / Delete.
[ ] Тема preview-вікна: білий фон, чорний текст.

Опціонально (потребує великого тестового файлу >25 MB):
[ ] Якщо вкласти файл >25 MB і натиснути Preview — побачите повідомлення
    "Failed to load file: 413: File is X MB; preview limit is 25 MB."
[ ] Download того ж файлу — побачите alert
    "Download failed (413): File is X MB; preview limit is 25 MB."
