# CHANGELOG

## Версія: excel embedded images (гілка feature/excel-images)

Дата: 2026-05-12

## Що змінилось

### Показ картинок, вбудованих у XLSX

Excel-файли часто містять вбудовані фото/картинки (логотипи, скани, фото
товарів). SheetJS, який рендерить таблицю, ці картинки повністю ігнорує —
тому раніше у Preview їх не було видно взагалі.

Тепер після відкриття XLSX-файлу всі вбудовані картинки з нього показуються
у вигляді галереї **під таблицею**. Картинки витягуються з ZIP-структури
файлу (xl/media/) на стороні браузера, окремих запитів до сервера немає.

Підтримувані формати картинок усередині XLSX: PNG, JPG/JPEG, GIF, BMP, WebP.

## Обмеження (свідомі компроміси)

1. Картинки показуються **під таблицею**, а не у тих клітинках де вони
   розташовані в оригінальному Excel. Точне позиціонування потребувало б
   парсингу xl/drawings/*.xml та мапінгу координат — це складно і часто
   ламається на різних варіантах експорту з Excel.

2. Якщо файл багатолистовий — показуються **всі картинки з робочої книги
   разом**, без розбивки по аркушах. Карта "аркуш → картинки" вимагає
   парсингу xl/worksheets/_rels/sheetN.xml.rels, що теж складно.

3. Працює тільки для ZIP-форматів: **xlsx, xlsm, xlsb**. Старий бінарний
   формат .xls та CSV не мають вбудованих картинок взагалі.

Якщо ці обмеження критичні — обговоримо складніший варіант з точним
позиціонуванням.

## Що тестувати на preview-URL

Стандартний потік (без регресій):
[ ] Звичайний xlsx БЕЗ картинок — Preview працює як раніше, нічого зайвого
    під таблицею не з'являється.
[ ] CSV-файл — Preview працює як раніше.
[ ] Bad-файл сф-1530_2 Олбрізсервіс 28.04.2026 (3).xlsx — відкривається швидко.
[ ] Багатолистовий xlsx — перемикання аркушів працює, галерея не дублюється.
[ ] Кнопки Download / Rename / Delete / меню "⋯" — все працює.
[ ] Тема preview — білий фон, чорний текст.

Нова функціональність:
[ ] xlsx з вбудованими картинками — під таблицею з'являється галерея,
    картинки видно.
[ ] Декілька картинок — всі показуються поруч, перенос на новий рядок
    автоматичний (flex-wrap).
[ ] Великі картинки масштабуються до max 300×300 px.
